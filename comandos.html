Comandos Linux

Conheça os comandos básicos para navegar e administrar o Linux.

Principais Comandos

ls

Lista arquivos e diretórios.

cd

Navega entre pastas do sistema.

mkdir

Cria novos diretórios.

rm

Remove arquivos ou pastas.

sudo

Executa tarefas como administrador.

X

ls

Exibe os arquivos e diretórios dentro da pasta atual.

ComandoFunção
lsLista arquivos
ls -lLista com detalhes
ls -aInclui arquivos ocultos
X

cd

Permite mudar de diretório dentro do sistema Linux.

  • cd /home → Vai para a pasta home
  • cd .. → Volta uma pasta
  • cd ~ → Vai para a pasta pessoal
X

mkdir

Cria novos diretórios.

  • mkdir projetos → Cria a pasta "projetos"
  • mkdir -p dir1/dir2 → Cria estrutura de pastas
X

rm

Remove arquivos e diretórios.

ComandoDescrição
rm arquivo.txtApaga um arquivo
rm -r pasta/Apaga uma pasta inteira
rm -f arquivo.txtForça a exclusão sem pedir confirmação
X

sudo

Executa comandos como superusuário (administrador).

  • sudo apt update → Atualiza pacotes
  • sudo reboot → Reinicia o sistema
  • sudo rm -r / → Apaga tudo
